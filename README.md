# 🏢 Tour Virtual 3D - BioTIC

Projeto desenvolvido durante o programa **SPRINT 2025**, focado na criação de uma plataforma web com um tour virtual 3D interativo para a BioTIC.

---

## 🎯 Objetivo do Projeto

Proporcionar uma experiência **imersiva e acessível**, permitindo que os usuários explorem os ambientes da BioTIC digitalmente através de uma interface web otimizada e responsiva.

---

## 🛠 Tecnologias e Ferramentas

* **Frontend:** React / Vite
* **Motor 3D/VR:** Marzipano / Three.js
* **Linguagem:** TypeScript / JavaScript
* **Estilização:** CSS / Tailwind

---

## 🚀 Como rodar o projeto localmente (Linux)

```bash
# Clone o repositório
git clone git@github.com:SEU_USUARIO/sprint2025_site.git

# Acesse a pasta do projeto
cd sprint2025_site

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```
